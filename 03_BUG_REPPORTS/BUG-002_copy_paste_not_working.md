--- BUG REPORT ---

Title:
[Inputs] - Copy-Paste Action - Value Not Inserted

Environment:
- OS: Windows 10
- Browser: Chrome
- Version: Latest

Precondition:
User is on Inputs page.

Steps to Reproduce:
1. Navigate to Inputs page.
2. Copy a numeric value (e.g., 123).
3. Click on the input field.
4. Paste using Ctrl + V.

Actual Result:
Pasted value is not inserted into the input field.

Expected Result:
Copied numeric value should be inserted into the input field.

Severity:
Minor

Priority:
P2

--- END BUG REPORT ---
