# Test Data Exploration — Inputs Module

## Valid Data
- 0 → Accepted
- Positive values → Accepted
- Negative values → Accepted

## Boundary Data
- Extremely large numbers → Accepted

## Invalid Data
- Alphabetic characters → Rejected
- Special characters (;?_=) → Rejected

## Additional Observations
- Multiple plus/minus signs are accepted (++, --)
- Mixed numeric and plus combinations accepted (e.g., 54+556)
