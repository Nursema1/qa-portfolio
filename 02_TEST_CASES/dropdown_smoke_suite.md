# TC-DROP-001 — Select Valid Option

Precondition:
Dropdown page is open

Steps:
1. Click dropdown
2. Select Option 1

Expected Result:
Selected option should be displayed

# TC-DROP-002 — No Selection

Precondition:
Dropdown page is open

Steps:
1. Do not select any option
2. Continue interaction

Expected Result:
System should remain stable without error

# TC-DROP-003 — Rapid Selection Change

Precondition:
Dropdown page is open

Steps:
1. Click dropdown
2. Rapidly switch between options

Expected Result:
System should update selection correctly without freeze
